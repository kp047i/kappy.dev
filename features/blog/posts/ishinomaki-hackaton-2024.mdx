10/12, 13, 14の3日間、石巻ハッカソン2024に参加してきました。今回が2回目です。石巻ハッカソンを主催している方とは、その方々がやっている別のイベントで個人スポンサーをしたきっかけで知り合うことができました。

## 今回のテーマ

今回の石巻ハッカソンのテーマが「かわいい」でした。自分は以前からメッセージを送るアプリを作りたいなと思っていたので、今回はリーダーとしてメンバーを募集してみるところからスタートしました。

## チームメンバー

自分含めて4人でチームを組むことになり、フロントからバックエンド、生成AI周りの知見が集まっている人が集まりました。自分はフロントエンドを担当しました。

## 技術スタック

今回使った技術スタックは以下の通りです。画像を永続化させるのにMySQLも使っています。

- フロントエンド
  - TypeScript
  - React
  - Remix
  - Tailwind CSS
  - shadcn/ui
- バックエンド
  - Go
- 生成AI
  - DALL·E 3

## 作ったもの

フレームの生成画面で、テーマや動物を選択しそれを元に生成された画像をもとに、そこにメッセージを入力して保存できるアプリを作成しました。メッセージのフォント、色、座標等は調節できるように入力のUIも作りました。

作られた画像たちはこのような感じです。

## まとめ

今回もその場で集まったメンバーで楽しくアプリを作り切ることができました。特に、UI周りに関しては褒めてもらえることが多く、shadcn/uiを使ってよかったなと思いました。また来年も参加します！
